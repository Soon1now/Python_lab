1. Что такое валидация данных?

Валидация данных — это процесс проверки данных на соответствие заданным правилам, ограничениям и критериям. Цель валидации — убедиться, что данные корректны, полны, согласованы и соответствуют ожидаемому формату и диапазону значений. Валидация необходима для предотвращения ошибок, обеспечения надёжности и целостности данных, а также для защиты от вредоносного ввода.


2. Как проверять строку на соответствие регулярному выражению?

re.fullmatch(pattern, string) 
pattern - содержит регулярное выражение, которое мы хотим проверить.
string - содержит строку, которую мы проверяем.


3. Какой модуль отвечает за работу с регулярными выражениями и какие у него есть методы?

Модуль, отвечающий за работу с регулярными выражениями, называется re. 

Основные методы модуля re:


re.search(pattern, string, flags=0): Ищет первое соответствие шаблону pattern в строке string. Возвращает объект match, если соответствие найдено, и None в противном случае.
re.match(pattern, string, flags=0): Пытается сопоставить шаблон pattern с началом строки string. 
re.fullmatch(pattern, string, flags=0): Пытается сопоставить шаблон pattern со всей строкой string. 
re.findall(pattern, string, flags=0): Находит все непересекающиеся совпадения шаблона pattern в строке string и возвращает их в виде списка строк.
re.finditer(pattern, string, flags=0): Находит все непересекающиеся совпадения шаблона pattern в строке string и возвращает итератор объектов match.
re.split(pattern, string, maxsplit=0, flags=0): Разбивает строку string на подстроки, используя шаблон pattern в качестве разделителя.

Методы re.search(), re.match() и re.fullmatch() возвращают объект match, если совпадение найдено. Объект match имеет следующие полезные методы:
match.group(num=0): Возвращает подстроку, соответствующую всей группе соответствия (если num=0), или определённой группе, захваченной скобками в регулярном выражении (если num — номер группы, начиная с 1).
match.start(group=0): Возвращает начальную позицию соответствия (или указанной группы) в строке.
match.end(group=0): Возвращает конечную позицию соответствия (или указанной группы) в строке.
match.span(group=0): Возвращает кортеж (start, end) с начальной и конечной позициями соответствия (или указанной группы).
