1 Описать 5 методов множеств.
- add(elem): 
	Добавляет элемент в множество. Если элемент уже существует, ничего не происходит.
- remove(elem):	
	Удаляет элемент из множества. Если элемент отсутствует, вызывает ошибку KeyError.
- discard(elem): 
	Удаляет элемент из множества. Если элемент отсутствует, ничего не происходит (без ошибки).
- pop(): 
	Удаляет и возвращает произвольный элемент из множества. Если множество пустое, вызывает ошибку KeyError.
- union(other) или | (оператор объединения): 
	Возвращает новое множество, объединяя текущее и другое множество.

3 Описать 5 методов словарей.
- get(key, default=None):
	Возвращает значение по указанному ключу. Если ключ отсутствует, возвращает значение по умолчанию.
- keys():
	Возвращает представление всех ключей словаря.
- values():
	Возвращает представление всех значений словаря.
- items():
	Возвращает представление всех пар "ключ - значение".
- pop(key, default=None):
	Удаляет ключ и возвращает его значение. Если ключ отсутствует, возвращает значение по умолчанию.
